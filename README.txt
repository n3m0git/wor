== Changelog ==

= Version 3.4.7 03 June 2016 =
- REVISED: Streamlined filter / infinite scroll and removed confusing options
- REVISED: Removed folio_fonts_url() function from add_editor_style as it was not working as intended

= Version 3.4.6 25 April 2016
- REVISED: Show the featured video when viewing single posts in the 'video' post format

= Version 3.4.5 =
- ADDED: Form styling adjustments for Gravity Forms, Jetpack Forms, Contact Forms 7, and Ninja Forms
- REVISED: Refactored code for better adherence to WordPress coding standards

= Version 3.4.3 =
- FIXED: Walker menu error if no custom menu was created
- Added: LinkedIn social icon

= Version 3.4.2 =
- Fixed: Post thumbnail sizing to reduce browser upscaling
- Added: Footer Copywrite option in WP Customizer
- Added: Homepage standard post type filter

= Version 3.4.1 =
- Fixed: lazy load effect on homepage posts
- Revised: homepage pagination to be always visible on scroll

= Version 3.4.0 =
- Bug Fix: Add loader before Magnific Popup is loaded to avoid direct linking
- Bug Fix: Remove special characters (exception: underscores, dashes) from titles when outputting classes to target Magnific Popup

= Version 3.3.0 =
- New Feature: Add dropdown support

= Version 3.2.0 =
- New Feature: Add standard post format option to media section
- New Feature: Option to load in pages on homepage or redirect to page instead.
- New Feature: Add descriptions to videos
- New Feature: Lazy Loading for non-paginated filter option

= Version 3.1.0 =
- Bug Fix: Fixed Categories to use slugs instead of names.
- Bug Fix: Fixed logo uploader in Customizer when using child themes.
- New Feature: Added loader for pages.

= Version 3.0.2.0 =
